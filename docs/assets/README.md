# Assets

Use this folder for public-safe screenshots or short demo clips.

Recommended files:

- `dashboard.png`
- `login.png`
- `target-modal.png`
- `settings.png`

Before adding media:

- Use fake targets or public test hosts only.
- Redact tokens, chat IDs, usernames and private infrastructure details.
- Avoid exposing browser extensions, local paths or terminal history.
